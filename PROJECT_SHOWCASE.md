## Product Summary

Lumière is a commercially packaged, single-restaurant website template designed for sale to restaurant owners worldwide. It delivers a Michelin-caliber digital presence with zero framework dependencies.

## Technical Highlights

- **Mobile-first** responsive design with 12-column CSS Grid
- **CSS Custom Properties** for easy theming
- **ES6 Modules** for clean JavaScript architecture
- **Intersection Observer** for performant scroll animations
- **Semantic HTML5** with ARIA accessibility
- **JSON-LD** structured data for restaurant SEO
- **Lazy loading** on below-fold images
- **Reduced motion** media query support
