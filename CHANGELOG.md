# Changelog

All notable changes to the Lumière Restaurant Template are documented in this file.

## [1.0.0] - 2026-06-15

### Added

- Initial release of Lumière Premium Restaurant Website Template
- Fullscreen cinematic hero section with parallax and dual CTAs
- About section with restaurant story and philosophy cards
- Signature dishes section with 4 premium dish cards
- Full menu preview with tabbed categories (Starters, Mains, Desserts, Drinks)
- Chef showcase section with portrait, quote, and awards
- Masonry gallery with lightbox (keyboard navigation support)
- Reservation form with client-side validation
- Testimonials section with 3 review cards
- Awards & recognition section
- Location section with Google Maps embed
- Newsletter subscription form
- Comprehensive footer with social links
- Mobile-first responsive design (12-column grid)
- CSS custom properties design token system
- Modular JavaScript architecture (ES6 modules)
- Scroll reveal animations with reduced motion support
- SEO optimization (meta tags, Open Graph, JSON-LD schema)
- Accessibility features (skip link, ARIA, semantic HTML)
- Lazy loading for below-fold images
- Centralized configuration file (`js/config.js`)
- 15 premium food and restaurant images
- SVG favicon
- Commercial documentation package
- MIT + Commercial LICENSE

### Technical

- HTML5 semantic markup
- CSS3 with `@import` modular architecture
- Vanilla JavaScript (no frameworks or dependencies)
- Google Fonts: Playfair Display + Inter
- Color palette: Primary #111111, Secondary #C8A45D, Background #F8F5F0
